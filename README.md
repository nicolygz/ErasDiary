# ErasDiary

ErasDiary é um aplicativo mobile inspirado no Letterboxd, porém voltado para shows e eventos musicais.

O app permite que usuários registrem os shows que já participaram, avaliem suas experiências, adicionem fotos e compartilhem com outros usuários em formato de feed.

O projeto foi desenvolvido com React Native utilizando Expo, com foco na construção da interface e experiência do usuário. As funcionalidades atuais estão implementadas no front-end com dados simulados, com previsão de integração futura com back-end para persistência e funcionalidades sociais completas.

Funcionalidades:

* Cadastro e autenticação de usuários (front-end)
* Registro de shows com avaliação e comentários
* Upload de imagens
* Feed/timeline de posts
* Interface inspirada em redes sociais

Tecnologias:

* React Native
* Expo
* AsyncStorage
* Expo Image Picker

Projeto acadêmico desenvolvido para fins educacionais.
